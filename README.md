# 📄 PDF AI 어시스턴트

PDF 강의 교재를 보면서 AI에게 바로 질문할 수 있는 웹 앱입니다.  
설치 없이 브라우저에서 바로 사용합니다.

🔗 **[바로 사용하기](https://HMSeodam.github.io/aitutor)**

---

## ✨ 주요 기능

- PDF 뷰어 (페이지 이동, 확대/축소)
- 텍스트 드래그 선택 후 해당 부분 AI 질문 (PC·모바일 모두 지원)
- 질문 범위 선택 — 선택 영역 / 현재 페이지 / 전체 PDF
- 빠른 질문 버튼 — 핵심 요약 / 개념 설명 / 용어 풀이 / 예상 시험 문제
- AI 이미지 생성 — "이미지로 표현해줘" 요청 시 자동 생성
- 연속 대화 및 긴 응답 지원 (최대 65,536 토큰)
- PC + 모바일 반응형

---

## 🚀 사용 방법

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 Gemini API 키를 발급받습니다.
2. 사이트 접속 후 **키 설정**에서 API 키를 입력합니다.
3. PDF 파일을 업로드합니다.
4. 텍스트를 드래그하거나 하단 입력창에 질문을 입력합니다.

---

## 🔒 개인정보 및 보안

- API 키는 브라우저 로컬 스토리지에만 저장됩니다.
- PDF 파일은 브라우저 내에서만 처리되며 서버에 업로드되지 않습니다.
- 질문 시 교재 텍스트가 Google Gemini API로 전송됩니다.
