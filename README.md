# 📄 PDF AI 어시스턴트

PDF 강의 교재를 브라우저에서 열람하면서, 교재 내용을 기반으로 Google Gemini AI에게 바로 질문할 수 있는 웹 애플리케이션입니다.

별도 설치나 서버 없이 **`index.html` 파일 하나**로 동작합니다.

🔗 **[바로 사용하기](https://사용자명.github.io/aitutor)**

---

## ✨ 주요 기능

- **PDF 뷰어** — 페이지 이동, 확대/축소, 드래그 앤 드롭 업로드
- **텍스트 드래그 선택** — 마우스(PC) 또는 손가락(모바일)으로 원하는 텍스트를 드래그하면 해당 부분에 대해 바로 질문 가능
- **질문 범위 선택** — 선택 영역 / 현재 페이지 / 전체 PDF 중 선택
- **AI 이미지 생성** — "이미지로 표현해줘", "그려줘" 등 요청 시 Gemini 2.5 Flash Image 모델로 실제 이미지 생성
- **빠른 질문 버튼** — 핵심 내용 요약 / 중요 개념 설명 / 어려운 용어 풀이 / 예상 시험 문제
- **연속 대화** — 이전 질문·답변 맥락을 유지하며 이어서 대화
- **긴 응답 지원** — 최대 65,536 토큰 출력, 응답이 잘린 경우 "이어서 계속 보기" 버튼 자동 표시
- **글자 크기 조절** — AI 답변 텍스트 크기 자유롭게 조절 (A- / A+)
- **PC + 모바일 완전 대응** — 반응형 레이아웃, 모바일에서 하단 시트 UI

## 🚀 사용 방법

### 1. API 키 발급

[Google AI Studio](https://aistudio.google.com/apikey)에서 Gemini API 키를 무료로 발급받습니다.

### 2. 접속 후 키 입력

사이트에 접속하면 API 키 입력 창이 뜹니다. 발급받은 키를 입력하고 **저장하고 시작**을 누릅니다.  
키는 브라우저 로컬 스토리지에만 저장되며 외부로 전송되지 않습니다.

### 3. PDF 업로드

**PDF 열기** 버튼을 누르거나 PDF 파일을 드래그 앤 드롭합니다.

### 4. 질문하기

| 방법 | 설명 |
|---|---|
| 텍스트 드래그 선택 | PDF 위의 텍스트를 드래그 → 팝업에서 **이 부분 질문하기** 클릭 |
| 빠른 질문 버튼 | 핵심 요약, 개념 설명, 용어 풀이, 시험 문제 버튼 클릭 |
| 직접 입력 | 하단 입력창에 자유롭게 질문 입력 후 Enter |

## ⚙️ 기술 스택

| 항목 | 내용 |
|---|---|
| AI 모델 | Google Gemini 2.5 Flash / Gemini 2.5 Flash Image |

| 프레임워크 | 없음 (순수 HTML / CSS / JavaScript) |

## 📦 GitHub Pages 배포 방법

1. 이 저장소를 Fork하거나 `index.html`을 새 저장소에 업로드합니다.
2. 저장소 **Settings → Pages** 로 이동합니다.
3. Branch: `main` / Folder: `/ (root)` 선택 후 **Save**를 누릅니다.
4. 1~2분 후 `https://사용자명.github.io/aitutor` 주소로 접속합니다.

- API 키는 사용자의 브라우저 로컬 스토리지에만 저장됩니다.
